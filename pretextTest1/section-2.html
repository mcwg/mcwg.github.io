Section 2 Initial Exploration: where does the method come from?

Subsection 2.1 (**) PENDING

NOTE: Have students do several examples of differentiation using the chain rule, and then write the corresponding antidifferentiation formulas. \(e^{3x}\)

NOTE: Emphasize that integration by substitution is just the reversal of the chain rule; the hard part is learning to recognize to which integrals we can apply this “reverse chain rule.”

NOTE: Possibly introduce the integral \(\int e^{x^2}dx\) and have students try to find the antiderivative using the guess-and-check method. Show how the obvious guess \(e^{x^2}\) does not work because of the extra term \(2x\) that appears when we apply the chain rule. Then point out if the \(2x\) from the chain rule had been under the integral sign in the first place, we could have easily done this integral. This gets the point across that integrands have to be in a special form to be integrated using substitution---a point you will need to come back to again and again.

NOTE: Then go on to do several more examples like \(\int t^2e^{t^3}dt\) and \(\int x^4\sqrt{x^5-4}\,dx\) by the guess-and-check method. Working through several examples this way leads naturally to the quest for a more systematic method, namely \(w\)-substitution.
