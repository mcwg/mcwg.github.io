Section 1 Steps to choosing a technique

¶
Step 1 : Is the integrand a derivative of a known function?

If so, this function is an antiderivative.

Step 2 : Is the integrand a product?

If so, is it of the form \(blob'\times f(blob)\text{?}\) The \(blob\) is any expression; and \(f\) is a function—with luck, a function you can integrate. The substitution of \(w=blob\) will work if you can integrate the function \(f\text{.}\)

Step 3 : Is the integrand a product?

If so, is it of the form \(u\cdot v'\text{?}\) If so, can you integrate \(v'\text{?}\) Then integration by parts may be useful.

Subsection 1.1 Examples

Example 1.1.
\(\displaystyle \int 3x^2\,dx\text{.}\)
Solution

Since \(3x^2\) is the derivative of \(x^3\text{,}\) we see that \(\displaystyle \int 3x^2\,dx = x^3 +C\text{.}\)

Example 1.2.
\(\displaystyle \int \cos(2t)\,dt\text{.}\)
Solution

Since \(\cos(2t)\) is the derivative of \(\frac{1}{2}\sin(2t)\text{,}\) we see that \(\displaystyle \int \cos(2t)\,dt = \frac{1}{2}\sin(2t) +C\text{.}\)

Example 1.3.
\(\displaystyle \int 3x^2(x^3+1)^4\,dx\text{.}\)
Solution

Since \(3x^2\) is the derivative of \(\displaystyle (x^3+1)\text{,}\) if \(blob =x^3+1\text{,}\) the integral can be written

\begin{equation*} \displaystyle \int 3x^2(x^3+1)^4\,dx = \int blob'(blob)^4\,dx. \end{equation*}

So we take

\begin{equation*} w=x^3+1\text{,} \end{equation*}

giving

\begin{equation*} dw=3x^2\,dx\text{.} \end{equation*}

Then we see that

\begin{equation*} \int 3x^2(x^3+1)^4\,dx = \int w^4\,dw. \end{equation*}

Fortunately, we can integrate

\begin{equation*} w^4\text{,} \end{equation*}

so we do that and substitute back into

\begin{equation*} x\text{:} \end{equation*}
\begin{equation*} \int 3x^2(x^3+1)^4\,dx = \int w^4\,dw = \frac{1}{5}w^5+C = \frac{1}{5}(x^3+1)^5+C. \end{equation*}

Note that if the

\begin{equation*} 3 \end{equation*}

had not been in the original integrand, we could still have used the same method. The only difference would be that

\begin{equation*} \displaystyle x^2\,dx = \frac{1}{3}\,dw \end{equation*}
Example 1.4.
\(\displaystyle\int t\cos t \, dt.\)
Solution

The integrand is a product, but it cannot be written in the form \(blob'\times f(blob)\text{.}\) However if we write \(u=t\) and \(v'=\cos t\text{,}\) so \(u'=1\) and \(v=\sin t\text{,}\) then

\begin{equation*} \int t\cos t \, dt=uv-\int u'v\, dt=t\sin t-\int 1\cdot\sin t\,dt. \end{equation*}

Since we can integrate \(\int\sin t\,dt =-\cos t +C\text{,}\) we can use integration by parts:

\begin{equation*} \int t\cos t \, dt=t\sin t-\int 1\cdot\sin t\,dt=t\sin t+\cos t +C. \end{equation*}
