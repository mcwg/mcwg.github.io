Solution 1.4.1.

The integrand is a product, but it cannot be written in the form \(blob'\times f(blob)\text{.}\) However if we write \(u=t\) and \(v'=\cos t\text{,}\) so \(u'=1\) and \(v=\sin t\text{,}\) then

\begin{equation*} \int t\cos t \, dt=uv-\int u'v\, dt=t\sin t-\int 1\cdot\sin t\,dt. \end{equation*}

Since we can integrate \(\int\sin t\,dt =-\cos t +C\text{,}\) we can use integration by parts:

\begin{equation*} \int t\cos t \, dt=t\sin t-\int 1\cdot\sin t\,dt=t\sin t+\cos t +C. \end{equation*}
