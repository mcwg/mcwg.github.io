Solution 1.3.1.

Since \(3x^2\) is the derivative of \(\displaystyle (x^3+1)\text{,}\) if \(blob =x^3+1\text{,}\) the integral can be written

\begin{equation*} \displaystyle \int 3x^2(x^3+1)^4\,dx = \int blob'(blob)^4\,dx. \end{equation*}

So we take

\begin{equation*} w=x^3+1\text{,} \end{equation*}

giving

\begin{equation*} dw=3x^2\,dx\text{.} \end{equation*}

Then we see that

\begin{equation*} \int 3x^2(x^3+1)^4\,dx = \int w^4\,dw. \end{equation*}

Fortunately, we can integrate

\begin{equation*} w^4\text{,} \end{equation*}

so we do that and substitute back into

\begin{equation*} x\text{:} \end{equation*}
\begin{equation*} \int 3x^2(x^3+1)^4\,dx = \int w^4\,dw = \frac{1}{5}w^5+C = \frac{1}{5}(x^3+1)^5+C. \end{equation*}

Note that if the

\begin{equation*} 3 \end{equation*}

had not been in the original integrand, we could still have used the same method. The only difference would be that

\begin{equation*} \displaystyle x^2\,dx = \frac{1}{3}\,dw \end{equation*}
