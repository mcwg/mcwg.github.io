Solution 3.1.1.

We can use substitution if the integrand has the form \(g'(x)f(g(x))\) and \(f\) is a function we can integrate. We have \(g(x) = x^4\text{,}\) so setting \(p(x) = g'(x)\) gives

\begin{equation*} p(x)= \frac{d}{dx} \left( x^4 \right) = 4x^3. \end{equation*}

Checking if \(p(x)=4x^3\) works, notice we can evaluate

\begin{equation*} \int p(x) e^{x^4} \,dx = \int 4x^3 e^{x^4} \,dx \end{equation*}

using substitution with \(w=x^4\text{.}\) Substitution can be applied if \(p(x)\) is a constant multiple of \(g'(x)=4x^3\text{,}\) so other answers are possible.
