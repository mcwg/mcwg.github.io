To determine whether it is continuous or differentiable, we need to determine whether the component functions are. The \(\vec{j}\) component function \(t^2-t\text{,}\) is continuous and differentiable everywhere. The \(\vec{i}\) component function is continuous and differentiable everywhere except possibly the point where the two pieces meet, when \(t=1\text{,}\) so we consider this point in detail.
