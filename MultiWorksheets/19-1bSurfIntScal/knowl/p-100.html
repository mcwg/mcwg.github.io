We now consider the integrals over \(S_1\) and \(S_2\text{:}\)

  1. \({y}\) is a positive function on all of \(S_1\text{,}\) so \(\displaystyle \int\int_{S_1}{y} \,dS\) is positive.

  2. \({y}\) is a negative function on all of \(S_2\text{,}\) so \(\displaystyle \int\int_{S_2}{y} \,dS\) is negative.
