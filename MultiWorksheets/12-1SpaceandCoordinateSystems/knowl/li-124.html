Item 5.1.1.
We can start by converting to Cartesian coordinates. We know \(r\sin{(\theta )}=y\) and \(r=\sqrt{x^2+y^2}\) so converting, we get \(y=\sqrt{x^2+y^2}\text{.}\) This is equivalent to \(y^2=x^2+y^2\) or \(x^2=0\text{.}\) Thus we get \(x=0\) which is the \(y\)-axis. Note however that since \(y= \sqrt{x^2+y^2}\) we can only have \(y\geq 0\) so this is the non-negative part of the \(y\)-axis. This matches Figure D.
