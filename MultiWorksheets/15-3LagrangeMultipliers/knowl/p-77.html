If instead of \(\lambda=1\) we have \(x = y\text{,}\) then \(x^2+y^2={25}\) implies that \(2x^2={25}\text{,}\) so \(x=y=\pm{5}/\sqrt{2}\text{.}\) Using \(x+y+z={5}\) to find the value of \(z\text{,}\) we find that the solutions to the system when \(x=y\) are \(\left({5}/\sqrt{2},{5}/\sqrt{2},{5}-{5}\sqrt{2}\right)\) and \(\left(-{5}/\sqrt{2},-{5}/\sqrt{2},{5}+{5}\sqrt{2}\right)\text{.}\)
