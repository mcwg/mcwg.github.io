The projection of this helix into the \(xy\)-plane will be a circle of radius \({3}\) centered at the origin, so we use \({3}\cos{(t)}\) and \({3}\sin{(t)}\) as components. When \(t=0\text{,}\) we want the curve to be at the point \(({-3},0,0)\text{,}\) so we choose functions in the \({3}\cos{(t)}\) for the \(\vec{i}\) component and \({3}\sin{(t)}\) as the \(\vec{j}\) component. For the \(\vec{k}\) component, we want the curve to move upward starting at \(0\) when \(t=0\text{,}\) so we can choose any increasing function with value \(0\) when \(t=0\text{.}\) One such function is \(t\text{.}\) Thus \(\vec{r}(t)={3}\cos{(t)}\vec{i} + {3}\sin{(t)}\vec{j}+ t\vec{k}\) is one possible parameterization.
