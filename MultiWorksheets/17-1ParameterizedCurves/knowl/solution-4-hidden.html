Recall that the graph of a function \(x=f(y)\) is parameterized by \(\vec{r}(t) =f(t)\vec{i} +t\vec{j}\text{,}\) so we get \(\vec{r}(t) =(t^2-1)\vec{i} +t\vec{j}\text{.}\)

We can test each point to see if it satisfies the parameterization we have derived.

  1. \(({0},{1})\) is on \(C\) since when \(y=t={1}\text{,}\) we have \(x=t^2-1={1}^2-1={0}\text{.}\)

  2. \(({3},{-2})\) is on \(C\) since when \(y=t={-2}\text{,}\) we have \(x=({-2})^2-1={3}\text{.}\)

  3. \(({1},{0})\) is not on \(C\) since when \(y=t={0}\text{,}\) we have \(x=t^2-1={0}^2-1={-1}\text{.}\)

  4. \(({-2},{3})\) is not on \(C\) since when \(y=t={3}\text{,}\) we have \(x={3}^2-1={8}\text{.}\)
