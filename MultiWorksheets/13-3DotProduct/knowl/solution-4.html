Solution 4.4.1.

We could solve this problem by finding the angle between \(\vec{v}\) and \(\vec{w}\text{,}\) then constructing the vector \(\vec{u}\) geometrically in the plane using the given angle. However, in this case, if we place \(\vec{v}\) and \(\vec{w}\) as position vectors in 2-space, we see that a vector \(\vec{u}\) making the same angle with \(\vec{w}\) can be obtained by reflecting the vector \(\vec{v}\) across the line \(y=-x\text{,}\) see the figure.

Since the head of \(\vec{v}\) is at the point \(({3.4},{2.8})\text{,}\) the head of \(\vec{u}\) is at \((-{2.8},-{3.4})\text{,}\) and hence \(\vec{u}={-2.8\mathit{\vec i}-3.4\mathit{\vec j}}\) is a vector that makes the same angle with \(\vec{w}\) but is not parallel to \(\vec{v}\text{.}\)
