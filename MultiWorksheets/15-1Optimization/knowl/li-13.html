Item.

The nature of a critical point can be determined using the Second Derivative Test. Specifically, let

\begin{equation*} H_f=\left[ \begin{array}{cc} f_{xx} \amp f_{xy} \\ f_{yx} \amp f_{yy} \end{array}\right] \end{equation*}

and let \(D=\det{(H_f)} \) (\(H_f \) is called the Hessian matrix of \(f(x,y) \)). If \((a,b) \) is a critical point of \(f(x,y) \text{,}\) then:

  1. if \(D(a,b)\gt 0 \) and \(f_{xx}\gt 0 \text{,}\) \((a,b) \) is a local minimum
  2. if \(D(a,b)>0 \) and \(f_{xx}\lt 0 \text{,}\) \((a,b) \) is a local maximum
  3. if \(D(a,b)\lt 0 \text{,}\) then \((a,b) \) is a saddle point (neither a maximum or a minimum)
  4. if \(D(a,b)=0 \text{,}\) the test is inconclusive.

Note: If the test is inconclusive or \(\nabla f \) is undefined, we need to use some other method.
