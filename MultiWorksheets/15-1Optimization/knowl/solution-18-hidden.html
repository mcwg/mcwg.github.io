A critical point of a function \(g(x,y)\) is a point where \(\nabla g(x,y)=\vec{0}\) or where \(\nabla g(x,y)\) is undefined. The partial derivatives

\begin{equation*} \newcommand{\amp}{\amp }\begin{aligned} g_x(x,y)\amp = {12}x+{12}y+{3} \\ g_y(x,y)\amp = {12}x-{12}y-{3}, \end{aligned} \end{equation*}

show that \(\nabla g(x,y)\) is defined everywhere, and \(\nabla g(x,y)=\vec{0}\) only at the points which are solutions to the system of equations

\begin{equation*} \newcommand{\amp}{\amp }\begin{aligned} {12}x+{12}y+{3} \amp = 0 \\ {12}x-{12}y-{3} \amp = 0. \end{aligned} \end{equation*}

If we add these two equations together, the \({12}y\) will cancel giving \({24}x=0\text{,}\) so \(x=0\text{.}\) Setting \(x=0\) in either of the two equations gives \(y={-{\textstyle\frac{1}{4}}}\text{.}\) Thus, the only solution to the system is \((x,y)=(0,{-{\textstyle\frac{1}{4}}})\text{.}\) Thus, the only critical point of \(g\) is \((0,{-{\textstyle\frac{1}{4}}})\text{.}\)

We compute

\begin{equation*} \newcommand{\amp}{\amp }D=\det \pmatrix{g_{xx} \amp g_{xy} \\ g_{yx} \amp g_{yy}}=\det \pmatrix{{12} \amp {12} \\ {12} \amp -{12}}=-{288}, \end{equation*}

and since \(\newcommand{\lt}{\lt }D\lt 0\) (at any point so in particular at \((0,{-{\textstyle\frac{1}{4}}})\)), we see that \((0,{-{\textstyle\frac{1}{4}}})\) is a saddle point.
