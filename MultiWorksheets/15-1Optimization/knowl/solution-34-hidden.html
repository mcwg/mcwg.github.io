A critical point of a function \(f(x,y)\) is a point where \(\nabla f(x,y)=\vec{0}\) or where \(\nabla f(x,y)\) is undefined. The partial derivatives

\begin{equation*} \begin{aligned} f_x(x,y)\amp = -{6}-2x\\ f_y(x,y)\amp = {3}-{4}y, \end{aligned} \end{equation*}

show that \(\nabla f(x,y)\) is defined everywhere, and \(\nabla f(x,y)=\vec{0}\) only when \((x,y)=({-3}, {{\textstyle\frac{3}{4}}})\text{.}\) Thus, the only critical point of \(f\) is \(({-3}, {{\textstyle\frac{3}{4}}})\text{.}\)

We compute \(D\)

\begin{equation*} D=\det \pmatrix{f_{xx} \amp f_{xy} \\ f_{yx} \amp f_{yy}}=\det \pmatrix{-2 \amp 0 \\ 0 \amp -{4}}={8}, \end{equation*}

and see that \(D\gt 0\) (at any point so in particular at \(({-3}, {{\textstyle\frac{3}{4}}})\text{,}\) so we see that \(({-3}, {{\textstyle\frac{3}{4}}})\) is a local maximum or a local minimum. To figure out which one it is, we can look at, for example, the concavity in \(x\)-direction by computing \(f_{xx}({-3}, {{\textstyle\frac{3}{4}}})\) which gives \(f_{xx}({-3}, {{\textstyle\frac{3}{4}}})=-2\lt 0\text{.}\) This indicates that \(f\) is concave down in the \(x\)-direction at \(({-3}, {{\textstyle\frac{3}{4}}})\) and so \(({-3}, {{\textstyle\frac{3}{4}}})\) is a local maximum.
