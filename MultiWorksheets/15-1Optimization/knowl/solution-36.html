Solution 5.2.1.

A critical point of a function \(g(x,y)\) is a point where \(\nabla g(x,y)=\vec{0}\) or where \(\nabla g(x,y)\) is undefined. The partial derivatives

\begin{equation*} \begin{aligned} g_x(x,y)\amp = {4}x+{8}y+{5} \\ g_y(x,y)\amp = {8}x-{8}y-{5}, \end{aligned} \end{equation*}

show that \(\nabla g(x,y)\) is defined everywhere, and \(\nabla g(x,y)=\vec{0}\) only at the points which are solutions to the system of equations

\begin{equation*} \begin{aligned} {4}x+{8}y+{5} \amp = 0 \\ {8}x-{8}y-{5} \amp = 0. \end{aligned} \end{equation*}

If we add these two equations together, the \({8}y\) will cancel giving \({12}x=0\text{,}\) so \(x=0\text{.}\) Setting \(x=0\) in either of the two equations gives \(y={-{\textstyle\frac{5}{8}}}\text{.}\) Thus, the only solution to the system is \((x,y)=(0,{-{\textstyle\frac{5}{8}}})\text{.}\) Thus, the only critical point of \(g\) is \((0,{-{\textstyle\frac{5}{8}}})\text{.}\)

We compute

\begin{equation*} D=\det \pmatrix{g_{xx} \amp g_{xy} \\ g_{yx} \amp g_{yy}}=\det \pmatrix{{4} \amp {8} \\ {8} \amp -{8}}=-{96}, \end{equation*}

and since \(D\lt 0\) (at any point so in particular at \((0,{-{\textstyle\frac{5}{8}}})\)), we see that \((0,{-{\textstyle\frac{5}{8}}})\) is a saddle point.
