Section 6 Fourth set: Getting the properties of logarithms straight

Is \(\log(a+b)=\log a+\log b\) a property of logarithms?

One can always test with values to see if equality holds. For example, using \(a=10\text{,}\) \(b=1\) we compute \(\log a+\log b=\log 10 + \log 1 = 1\text{.}\) Since this is not the value of \(\log(a+b)=\log 11\text{,}\) we now know that \(\log(a+b)=\log a+\log b\) is not true in general.

Regretfully, there are plenty of false equations, like \(\log A \log B=\log A+\log B\) (which is false!) that look similar to true properties, so one must get lots of practice using the real properties and keep in mind that one can always test some values.
