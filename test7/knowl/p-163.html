Regretfully, there are plenty of false equations, like \(\log A \log B=\log A+\log B\) (which is false!) that look similar to true properties, so one must get lots of practice using the real properties and keep in mind that, when in doubt, one can always test some values. For example:

Is \(\log(a+b)=\log a+\log b\) a property of logarithms?
We can test this with \(a=10\text{,}\) \(b=1\text{.}\) We compute \(\log a+\log b=\log 10 + \log 1 = 1\text{.}\) Since this is not the value of \(\log(a+b)=\log 11\text{,}\) we now know that \(\log(a+b)=\log a+\log b\) is not true in general, so it is not a property!
