Solution 5.1.1.

We are solving for an exponent, so we use logarithms. Since the base is the number \(e\text{,}\) it makes the most sense to use the natural logarithm. Using the log rules, we have

\begin{equation*} \newcommand{\amp}{\amp }\begin{aligned} e^{{{0.13}}x} \amp= {150}\\ \ln (e^{{{0.13}}x}) \amp= \ln({150})\\ {{0.13}}x \amp= \ln({150})\\ x \amp= {{\ln({150})} \over {{{0.13}}}} = {38.543}. \end{aligned} \end{equation*}
