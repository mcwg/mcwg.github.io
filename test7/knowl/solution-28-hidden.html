We are solving for an exponent, so we use logarithms. We can use either the common logarithm or the natural logarithm. Since \({5}^{{1}} = {5}\) and \({5}^{{2}} = {25}\text{,}\) we know that \(x\) must be between 1 and 2. Using the log rules, we have

\begin{equation*} \newcommand{\amp}{\amp }\begin{aligned} {5}^x \amp= {17}\\ \log ({5}^x) \amp= \log({17})\\ x \log({5}) \amp= \log({17})\\ x \amp= {{\log({17})} \over {\log({5})}} = {1.76}. \end{aligned} \end{equation*}

If we had used the natural logarithm, we would have

\begin{equation*} x = {{\ln({17})} \over {\ln({5})}} = {1.76}. \end{equation*}
