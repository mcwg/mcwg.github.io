Item.

One caption for all the figures. First create a file (say "manyfig.tex") containing
\valign[b]
{
{\input{figa}}*
{\input{figb}}/
{\input{figc}}
}
Here
  • \valign aligns the figures.
  • [b] makes them bottom-aligned. You can use [c] (centered) and [t] (top-aligned) also.
  • * means put in some space. You can use ** (which means put in lots of space) also.
  • / means make a new line of figures.
Then use
\begin{fakefig}
\finput{manyfig}{Caption}
\end{fakefig}
