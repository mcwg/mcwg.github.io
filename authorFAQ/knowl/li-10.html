Item 10.

Install Python using homebrew with pyenv from the terminal run (hit enter between lines)
brew install pyenv
pyenv install 3.10
pyenv global 3.10
Then add the following lines to your .bash_profile file:
# pyenv setup
export PYENV_ROOT="$HOME/.pyenv"
command -v pyenv >/dev/null || export PATH="$PYENV_ROOT/bin:$PATH"
eval "$(pyenv init -)"
Close the Terminal window and reopen it for all changes to have effect.
