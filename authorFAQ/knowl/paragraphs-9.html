Case 1: The problem asks about a single set of figures that are similar to each other in purpose.

For example:
  • Which of Figures (I)--(IV) is a parabola?
  • Match Figures (I)--(IV) to statements (a)--(d).
  • For each Figures (a)--(d). what critical points can you see?
Note that, in general, if the figures make up parts of a problem, the figures should be (a)--(d), so that the solution can have parts (a)--(d).
In Case 1, we give the figures upper-left-hand corner labels, which happens in the filenamefig.tex file in the line
\begin{aspic}|1|[(a)](2,2)
