Item.

When you want to input a figure with number but no caption into a file named "filename.tex" insert
\begin{stayfig}
\finput*{filenamefig}
\end{stayfig}
where you want the figure to appear. Here "filenamefig.tex" is the file containing the information to input the figure. Notice the filename has "fig" appended to the corresponding filename ("filename.tex").
If you want to reference this figure, use \fref{filenamefig}.
