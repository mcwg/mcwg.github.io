The critical points break up the \(x\)-axis into the four intervals \((-\infty, 15)\text{,}\) \((15, 17)\text{,}\) \((17, 21)\) and \((21, \infty)\text{.}\) Since a function can only change direction at a critical point, to determine whether \(g\) is increasing or decreasing on each of these intervals, we only need check whether \(g'\) is positive or negative at one point in each interval. We have:
  1. \(g'(14)\gt0\text{,}\) so \(g\) must be increasing on \((-\infty , 15)\text{.}\)
  2. \(g'(16)\gt0\text{,}\) so \(g\) must be increasing on \((15 , 17)\text{.}\)
  3. \(g'(18)\lt0\text{,}\) so \(g\) must be decreasing on \((17 , 21)\text{.}\)
  4. \(g'(22)\gt0\text{,}\) so \(g\) must be increasing on \((21 , \infty)\text{.}\)
