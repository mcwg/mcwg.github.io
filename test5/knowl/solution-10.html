Solution 5.2.1.

  1. We have \({\rm curl} (\vec{F})(x,y,z)= (0 - 0)\vec{i} - (0 - 0)\vec{j} + (-1 - 0)\vec{k} = -\vec{k}\text{,}\) and thus \({\rm curl} (\vec{F})(0,0,0)=-\vec{k}\text{.}\)

  2. We have \({\rm div} (\vec{F})(x,y,z)= 0 + 0 + 0 = 0\text{,}\) and thus \({\rm div} (\vec{F})(0,0,0)=0\text{.}\)
