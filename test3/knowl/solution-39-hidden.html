In addition to needing to find \(v\) from \(v'\text{,}\) we need to be able to find \(\int u'v\,dx\text{.}\) For instance, using integration by parts to find \(\displaystyle \int e^{-x^2}\cos x\,dx,\) we can let \(\displaystyle u=e^{-x^2}\) and \(\displaystyle v'=\cos x.\)

This gives

\begin{align*} u'\amp =-2xe^{-x^{2}}\\\ v\amp =\sin x\\\ \mbox{so}\quad \int \underbrace{e^{-x^2}}_{u} \cdot \underbrace{\cos x}_{v'}\,dx \amp = \underbrace{e^{-x^2}\cos x}_{uv} -\int \underbrace{-2xe^{-x^{2}}\sin x }_{u'v}\,dx. \end{align*}

But since we are unable to find \(\displaystyle \int2xe^{-x^{2}}\sin x,\) we see that knowing \(v\) alone is not sufficient. We must also be able to find \(\displaystyle \int u'v\,dx.\)
