Solution 3.1.1.

  1. This matches (i). Given \(u=x\text{,}\) \(v'=\cos x\text{,}\) we have \(u'=1\text{,}\) \(v=\sin x\text{:}\)

    \begin{equation*} \begin{aligned} \int \underbrace{x}_{u}\underbrace{\cos x}_{v'}\,dx \amp = \underbrace{x\sin x}_{uv}-\int\underbrace{1\cdot\sin x}_{u'v}\,dx\\ \text{so}\quad\int x\cos x\,dx \amp = x\sin x-\int\sin x\,dx. \end{aligned} \end{equation*}

  2. This matches (iii). Given \(u=x\text{,}\) \(v'=e^{2x}\text{,}\) we have \(u'=1\text{,}\) \(v=0.5e^{2x}\text{:}\)

    \begin{equation*} \begin{aligned} \int \underbrace{x}_{u}\underbrace{e^{2x}}_{v'}\,dx \amp = \underbrace{x\cdot0.5xe^{2x}}_{uv}-\int\underbrace{1\cdot0.5e^{2x}}_{u'v}\,dx\\ \text{so}\quad\int xe^{2x}\,dx\amp =0.5xe^{2x}-\int0.5e^{2x} \,dx. \end{aligned} \end{equation*}

  3. This matches (ii). Given \(u=\cos x\text{,}\) \(v'=x\text{,}\) we have \(u'=-\sin x\text{,}\) \(v=0.5x^{2}\text{:}\)

    \begin{equation*} \begin{aligned} \int \underbrace{\cos x}_{u}\underbrace{x}_{v'}\,dx \amp = \underbrace{\cos x\cdot0.5x^{2}}_{uv}-\int\underbrace{-\sin x\cdot0.5x^{2}}_{u'v}\,dx\\ \text{so}\quad\int x\cos x\,dx \amp = 0.5x^2\cos x+\int0.5x^2\sin x\,dx. \end{aligned} \end{equation*}

  4. This matches (iv). Given \(u=e^{2x}\text{,}\) \(v'=x\text{,}\) we have \(u'=2e^{2x}\text{,}\) \(v=0.5x^{2}\text{:}\)

    \begin{equation*} \begin{aligned} \int \underbrace{e^{2x}}_{u}\underbrace{x}_{v'}\,dx \amp = \underbrace{e^{2x}\cdot0.5x^2}_{uv}-\int\underbrace{2e^{2x}\cdot0.5x^2}_{u'v}\,dx\\ \text{so}\quad\int xe^{2x}\,dx \amp = 0.5x^2e^{2x}-\int x^2e^{2x} \,dx. \end{aligned} \end{equation*}
